# Crane — Development Estimation Breakdown

| Module | Task | Description |
|--------|------|-------------|
| **Setup & Infra** | Project scaffolding & architecture | Monorepo setup (Next.js frontend + Node.js backend), TypeScript config, data schemas (Company, Fund, Flag, Todo, Activity, MonthlyFinancials), env config, linting. Architecture doc with API contracts. |
| **Setup & Infra** | Cloud infra & CI/CD | Vercel (frontend), AWS/GCP (backend API + DB). PostgreSQL for relational data, Redis for caching/sessions, S3 for document storage. CI/CD pipeline with staging + prod. Monitoring, logging, error tracking (Sentry). |
| **Setup & Infra** | Database schema & migrations | Core tables: companies, funds, financials, flags, todos, activity_events, commentary, users, teams. Foreign keys, indexes, RLS policies. Migration framework (Prisma/Drizzle). Seed script with realistic demo data. |
| **Auth & Users** | Auth system + RBAC | Login/logout (JWT or session-based), SSO integration (Google Workspace), forgot/reset password. Role-based access: Principal, Partner, Associate, Analyst, Observer. Permission matrix per role. |
| **Auth & Users** | Multi-owner company model | Company ownership as many-to-many (1–3 owners per company). Observer role for visibility without edit rights. Dashboard filtering by ownership + observation. Team member profiles with avatars. |
| **Data Connectors** | Attio CRM integration | OAuth connection to Attio. Sync company contacts, founder interactions, deal pipeline, meeting history. Bi-directional: push notes/activity back to CRM. Polling every 2 hours. Field mapping config. |
| **Data Connectors** | Dropbox document ingestion | OAuth connection to Dropbox. Monitor per-company folders (IC Papers, Board Decks, Financials, Legal). Detect new/updated files. Extract metadata (filename, date, type). Store references + thumbnails. Polling every 6 hours. |
| **Data Connectors** | Gmail integration | OAuth connection to Gmail. Scan for founder update emails, investor communications. Extract key data points from email body (revenue mentions, hiring updates). Thread tracking. Polling every 30 minutes. |
| **Data Connectors** | Notion wiki sync | OAuth connection to Notion. Sync portfolio database pages, meeting notes, investment memos. Map Notion properties to internal schema. Polling every 24 hours. |
| **Data Connectors** | Granola meeting transcripts | API integration with Granola. Ingest call transcripts, extract action items, sentiment signals. Map to company activity timeline. Webhook-based or polling every 24 hours. Error handling for connection failures. |
| **Data Connectors** | Specter company signals | API integration with Specter. Monitor web traffic changes, hiring signals, funding rounds, competitor news. Map signals to company alerts. Polling every 12 hours. |
| **Data Connectors** | Fund accounting system | Integration with fund admin accounting system. Import NAV, cost basis, carrying values, MoIC per company. Weekly sync. Read-only — this is the source of truth for fund-level numbers. |
| **Data Connectors** | AlphaSense market research | API integration with AlphaSense. Pull market intelligence, research reports, competitor analysis. Map to Intelligence Hub. Optional Phase 2 connector. |
| **Data Connectors** | Connector management UI | Settings page: connector cards with status (Connected/Error/Disconnected), last sync time, record count. Configure button for field mapping. Fix Connection flow for error states. Sync frequency config. |
| **Data Connectors** | Data conflict resolution engine | When multiple sources report different values for the same metric and date, flag discrepancy. Priority hierarchy: Fund Accounting > Founder-validated > Board deck (OCR) > Email > External signals. Human-in-the-loop validation UI with side-by-side comparison. Orange highlight in review workflow. |
| **Dashboard** | Main dashboard page | Personal dashboard for logged-in user. Urgent cards (Needs Attention count, Overdue Actions count). Info cards (My Companies, Upcoming Boards, Next Review). Critical Alerts section with actionable dropdowns. |
| **Dashboard** | My To-Dos section | To-do list with priority badges, company pills, due dates, overdue highlighting. Source tracking (manual vs. converted from flag). Add new to-do modal. Mark complete toggle. Persisted in database. |
| **Dashboard** | Board Prep cards | Upcoming board meetings with countdown (days away). Quick actions: Start Prep, View Company. Sorted by nearest date. Shows company name, board date, fund. |
| **Dashboard** | Activity & Alerts feed | Unified feed combining system-generated alerts (flags) and activity events. Sorted by timestamp (newest first). Alerts have action dropdowns (Convert to To-Do, Log Note, Resolve, Snooze). Activity shows type badge, description, severity icon. |
| **Dashboard** | My Companies carousel | Horizontal scrollable cards for owned companies. Each card: company avatar, name, RAG status, MRR with trend arrow, sparkline chart, last updated date. Stale data highlighting (30d+ without update). Click navigates to company detail. |
| **Command Center** | Portfolio summary strip | Top-level stats: Active Core count, Green/Amber/Red counts, Average Runway, Recently Updated (30d), Stale (30d+). Clickable to filter the table below. Fund filter dropdown (Fund I/II/III) filters all data on page. |
| **Command Center** | Compact Action Matrix | 3x3 grid: Health (On Track / At Risk / Underperforming) x Upside (High Potential / Emerging / Limited Potential). Each cell shows company count and action label (Lean In / Anticipate / Watch / De-prioritise). Clickable to filter table. Manual team placement, not auto-calculated. |
| **Command Center** | Portfolio table | Full company table: name, fund, cost, carrying value, MoIC, RAG badge, ARR, cash balance, net burn, runway (red if < 6mo), fundraising status. Sortable columns. Active/Exited/All toggle. Multi-filter bar (Health, Upside, Stage, Fund, Owner). Export XLSX button. |
| **Action Matrix** | Full Action Matrix page | Expanded 3x3 matrix with company cards in each cell. Per-card: company name, stage, MRR, runway, flag count. Summary stats at top (Active count, Flag count, Avg Runway). Filter pills (Lean In / Anticipate / Watch / De-prioritise / All). Fund filter. |
| **Action Matrix** | Manual placement mechanism | Action Matrix placement is a team judgment call, not algorithmic. UI for team to drag-and-drop or select Health + Upside per company during monthly reviews. Store placement with timestamp and who made the call. History of placement changes. |
| **Company Detail** | Company overview tab | Company header: name, stage, sector, fund badges. Action buttons (Log Note, To-Do, Check-In, Board Prep, Attio). Cost/Carrying/MoIC strip. RAG history dots (last 4 quarters). Alert banner with Take Action dropdown. Tab navigation. |
| **Company Detail** | Company details & timeline | Company Details section (legal name, location, industry, website, management, Crane lead, currency, region). Activity Timeline (chronological events with type badges, dates, descriptions). Description and Recent Progress text blocks. |
| **Company Detail** | Key Concerns & Action Points | Editable lists: Key Concerns (numbered items) and Action Points (numbered items). These feed into the quarterly LP report commentary. Team members can add/edit/remove items. Change tracking with timestamps. |
| **Company Detail** | Metrics & Charts tab | Financial charts: MRR trend (line), ARR growth (bar), burn rate (area), runway projection. Diversity metrics: headcount breakdown (male/female/ethnic minority for FTE and Board). Quarterly comparison tables. Data sourced from MonthlyFinancials records. |
| **Company Detail** | Documents tab | List of ingested documents from Dropbox. Filterable by type (Board Deck, IC Paper, Legal, Financials). Preview thumbnails. Upload date, source indicator. Link to original in Dropbox. |
| **Company Detail** | Notes & Actions tab | Full note history for this company. Log Note modal (text, category, date). To-dos specific to this company. Check-in scheduling. All persisted and visible to team. |
| **Company Detail** | Fundraising tab | Equity fundraising status, debt fundraising status, burn reduction actions, near-term exit flag. Editable fields. Timeline of fundraising events. Runway projection with scenarios. |
| **Portfolio Review** | Monthly Internal Review | 30–45 min team review workflow. Step through companies in priority order (red first, then amber, then green). Inline note-taking per company. RAG status update. Quick flag creation. Mark reviewed/skipped. Progress tracker. |
| **Portfolio Review** | Quarterly LP Report workflow (V1) | 4-step workflow: (1) Founder Form — founders fill validated data via pre-populated form replacing Gigs Excel, (2) Team Review — team reviews submitted data and adds qualitative input, (3) Team Commentary — RAG status, summary, key concerns, action points per company, (4) Export — generate Asset Metrix XLSX (primary) + optional LP Report PDF. |
| **Portfolio Review** | Founder validation portal | External-facing form sent to founders. Pre-populated with last known values. Fields: Revenue, COGS, EBITDA, Cash Balance, Burn, Headcount (with diversity split), ARR, Bookings, Customer Count, Net Retention. Founders accept, edit, or flag values. Replaces Gigs Excel plugin. Mobile-responsive. |
| **Portfolio Review** | Monthly-to-quarterly AI summarization | Monthly commentary accumulates over 3 months. At quarter-end, AI condenses 3 months of notes into quarterly LP narrative per company. Team reviews and edits AI draft. Saves significant manual writing time. Structured prompt with company context. |
| **Portfolio Review** | Review history | Log of all completed reviews (monthly and quarterly). Date, type, companies reviewed, number of changes. Filterable list. Click to view snapshot of that review's commentary. |
| **Board Prep** | Board prep generator | Select company + upcoming board. AI-generated brief based on latest board deck, email updates, call transcripts, and platform data. Sections: Company Snapshot (health, MRR, runway, customers), Changes Since Last Board, Key Changes list, Active Flags, Suggested Discussion Points. |
| **Board Prep** | Board prep sharing & export | Share button for distributing prep to team. Export PDF for offline use. Draft status tracking (Draft / Not yet shared / Shared). Last refreshed timestamp. Prepare for multiple companies with different board dates. |
| **Alert Engine** | Alert rule framework | Configurable alert rules with thresholds: Runway Alert (< X months), Burn Acceleration (> X% QoQ), Engagement Gap (no contact for X days), Growth Inflection (MRR growth > X% MoM), Key Departure, Board Coming Up, Fundraising Signal, Market Signal, Pivot Signal. Enable/disable per rule. |
| **Alert Engine** | Alert generation & delivery | Background job evaluates rules against latest data. Generates Flag records with: company, type, headline, explanation, suggested action, urgency (high/medium/low). In-app notification badges. Optional Slack integration for high-urgency alerts. Snooze and dismiss actions. |
| **Alert Engine** | Alert-to-action conversion | Convert flag to to-do (one click). Log note and resolve. Mark as actioned. Snooze for 7 days. Bulk actions on multiple flags. Track resolution rate and response time for team analytics. |
| **Export & Reporting** | Asset Metrix XLSX export | Multi-sheet XLSX matching Asset Metrix template structure. Sheet 1: Company Commentary (ID, name, date, year, quarter, RAG, runway, summary, progress, concerns, actions). Sheet 2: Company Static (ID, name, location, industry, website, management, lead partner, description). Sheet 3: Company Periodic (ID, name, date, revenue, EBITDA, COGS, margins, cash, burn, ARR, headcount, diversity). 6-row header format (technical names, types, blanks, friendly names). |
| **Export & Reporting** | LP Report PDF | Full quarterly report matching Crane I Manager's Report format. Sections: Cover Page, Standing Data, Manager's Commentary (NAV Waterfall chart, TVPI trend, portfolio overview table, follow-on investments, NAV uplift/write-down, Uses of Funds, Geographic Distribution), Company Snapshots (per-company RAG history, metrics, assessment), Company Data Pages (financials, diversity). Print-optimized CSS with A4 page breaks. |
| **Export & Reporting** | Gigs Performance Data CSV | Per-company CSV matching Gigs template: Investment Ref, Category 1/2, Currency, Financial Indicator, then monthly columns. Rows: Revenue (core + other), COGS, R&D, Sales & Marketing, G&A, EBITDA, Cash, ARR, Bookings, Customers, Net Retention, Burn, Headcount (male/female/ethnic minority for FTE and Board). |
| **Export & Reporting** | Fund Summary CSV | Fund standing data (name, currency, vintage, committed, deployed, available, TVPI, DPI, NAV). TVPI History table. NAV Waterfall. Uses of Funds. Geographic Distribution. Internal use for fund-level analysis. |
| **Settings** | Data Sources management | Connector cards grid with status, last sync, record count. Configure modal for field mapping and sync frequency. Fix Connection flow for error states. Add new connector flow. |
| **Settings** | Team management | Team member list with roles (Principal, Partner, Associate). Invite new member flow. Role assignment. Avatar upload. Activity log per team member. |
| **Settings** | Fund management | Fund configuration: name, vintage, currency, accounting basis. No fund performance metrics in platform (goes to Asset Metrix directly). Fund used only for filtering and grouping companies. |
| **Settings** | Alert Rules configuration | Configurable threshold cards per alert type. Enable/disable toggles. Edit threshold values. Note about iterative calibration. Preview of which companies would currently trigger each rule. |
| **Settings** | Data Conflict Resolution config | Source priority hierarchy display and configuration. Confidence scoring rules. Discrepancy threshold (what % difference triggers a conflict flag). Resolution audit log. |
| **Settings** | Portfolio config | Default views, sorting preferences. Gigs field mapping (which internal fields map to which Gigs template columns). LP Report template selection. Asset Metrix field mapping. |
| **Intelligence Hub** | Market intelligence dashboard | *(Phase 2)* Aggregated market signals from Specter, AlphaSense, news feeds. Per-company signal timeline. Sector-level trends. Competitor funding/hiring alerts. Saved searches. |
| **Search & Discovery** | Semantic search | *(Phase 2)* Full-text search across all ingested data: documents, emails, transcripts, notes, financials. Elasticsearch-powered. Filters by company, date range, source type, document type. Search result snippets with highlighting. |
| **QA & Testing** | Unit tests | Backend: data transformation logic, alert rule evaluation, SOV-style calculations, export generation (XLSX structure, CSV format), deduplication, conflict detection. Frontend: component rendering, filter logic, workflow state transitions. |
| **QA & Testing** | Integration tests | API connector mocks + live sandbox tests. Database migration validation. Auth flow end-to-end. Export file structure validation (open generated XLSX, verify sheets/columns). Founder portal form submission. |
| **QA & Testing** | E2E tests | Critical flows: login → dashboard → company detail → log note. Monthly review workflow end-to-end. Quarterly workflow: founder form → team review → commentary → export. Alert generation → conversion to to-do → completion. Board prep generation. |
| **QA & Testing** | Report accuracy validation | Compare generated Asset Metrix XLSX against template structure. Verify LP Report PDF sections and data accuracy. Cross-check exported financials against source data. Diversity metric calculations. |
| **QA & Testing** | Performance & load testing | Dashboard load time with 50+ companies. Export generation time for full fund (17 companies x 12 months financials). Concurrent user testing (5 team members). Connector sync performance under rate limits. |
